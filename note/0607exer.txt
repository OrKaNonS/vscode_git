DOM 실습

1) domex1.heml, domex1.js
R, G, B 3가지 값(0~255)을 입력하면 배경색을 변경


2) domex1.heml, domex1.js
값을 2개 입력받고 +, -, *, / 버튼을 클릭하면 연산하여 결과 출력

3) domex3.heml, domex3.js
canvas(640*480)를 하나 만들고
한점과 가로, 세로 입력 후 사각형 버튼 클릭시 사각형 그리기
한점과 반지름 입력 후 원 버튼 클릭시 원 그리기

3) domex4.heml, domex4.js
한점과 가로, 세로 입력 후 사각형 버튼 클릭시 SVG 사각형 그리기
한점과 반지름 입력 후 원 버튼 클릭시 SVG 원 그리기
